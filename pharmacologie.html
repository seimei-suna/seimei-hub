Pharmacologie

Section Médicale Seimei · Sunagakure

La pharmacologie est l’étude des substances utilisées pour soigner. À Sunagakure, nous combinons les remèdes traditionnels issus de notre registre botanique avec les connaissances médicales avancées.

1

Antidouleurs (Analgésiques)

Saule blanc
(Salix alba)
Remède

Effet similaire à l’aspirine. Anti-douleur léger et anti-inflammatoire naturel extrait de l’écorce.

Usage : Douleurs musculaires, maux de tête, fièvre légère.

Pavot somnifère
(Papaver somniferum)
Remède

Analgésique puissant dérivé de l’opium. Réservé aux cas les plus graves en raison de sa puissance.

Usage : Douleurs sévères post-opératoires, blessures de combat graves.

Risque de dépendance. Réservé aux cas graves. Dosage strict obligatoire.
Anesthésie locale par chakra
Technique

Le médecin ninja peut engourdir une zone ciblée en perturbant les signaux nerveux avec son chakra.

Usage : Avant sutures, petites interventions chirurgicales.

2

Anti-inflammatoires

Aloe vera
(Aloe vera)
Spécialité Suna

Gel anti-inflammatoire naturel extrait des feuilles. Remède polyvalent du désert, abondant à Sunagakure.

Usage : Brûlures légères, irritations cutanées, cicatrisation.

Cactus / Opuntia
(Figuier de Barbarie)
Spécialité Suna

Protection gastrique et hydratation. Ses fibres mucilagineuses forment un gel protecteur pour l’estomac.

Usage : Inflammation digestive, complément nutritionnel en milieu désertique.

Cataplasme froid
Remède

Mélange d’herbes broyées appliqué sur une zone enflammée. Préparation simple accessible à tous les grades.

Usage : Entorses, hématomes, gonflements.

3

Antipoisons

Dokunoki no Jutsu
(Technique d’Extraction de Poison)
Technique

Extraction directe du poison par bulle de chakra. Le traitement le plus efficace contre les empoisonnements, mais requiert le grade Adepte minimum.

Usage : Tout type d’empoisonnement. Traitement de référence.

Charbon actif
Remède

Absorbe les toxines dans l’estomac si le poison a été ingéré récemment (moins de 2 heures).

Usage : Empoisonnement par ingestion. Administration orale immédiate.

Quinquina
(Cinchona officinalis)
Remède

Antiparasitaire naturel. La quinine extraite de son écorce neutralise les toxines parasitaires.

Usage : Piqûres d’insectes venimeux, infections parasitaires.

Artemisia annua
(Armoise annuelle)
Remède

Antiparasitaire reconnu dans toutes les nations. L’artémisinine combat les parasites transmis par les insectes.

Usage : Piqûres, fièvres parasitaires.

4

Anesthésiants

Pavot somnifère (dose contrôlée)
(Papaver somniferum)
Remède

Sédation légère pour les interventions chirurgicales. Le dosage doit être calculé avec précision selon le poids du patient.

Usage : Anesthésie générale légère pour interventions de moyenne durée.

Dosage critique — surdosage = arrêt respiratoire. Supervision Adepte obligatoire.
Esmolol
Remède

Agent cardiaque ralentisseur utilisé en chirurgie pour contrôler le rythme cardiaque pendant les opérations délicates.

Usage : Chirurgie cardiaque, opérations complexes. Réservé aux Experts.

Réservé au grade Expert. Manipulation incorrecte = risque d’arrêt cardiaque.
Engourdissement par chakra
Technique

Technique de base : le médecin ralentit la transmission nerveuse dans une zone ciblée. Moins efficace qu’une anesthésie chimique mais sans effets secondaires.

Usage : Petites interventions, complément d’anesthésie chimique.

5

Cicatrisants

Aloe vera
(Aloe vera)
Spécialité Suna

Accélère la cicatrisation, hydrate et protège la peau. Remède de base pour les plaies superficielles.

Usage : Plaies superficielles, brûlures légères, soins post-opératoires cutanés.

Paume Spirituelle
(Technique Aspirant)
Technique

Régénération des tissus superficiels par chakra. Première technique de soin apprise par les médecins ninjas.

Usage : Cicatrisation accélérée des plaies légères et écorchures.

Paume Mystique
(Technique Expert)
Technique

Régénération cellulaire avancée. Permet de refermer les plaies profondes et de réparer les tissus internes.

Usage : Blessures profondes, lacerations, réparation tissulaire post-chirurgicale.

Miel médical
Remède

Antiseptique naturel, favorise la cicatrisation en milieu humide. Empêche la prolifération bactérienne sur les plaies ouvertes.

Usage : Pansements de plaies chroniques, brûlures étendues.
